# VelaraLoop — AI-Augmented Product Delivery Framework

> Ship real products, not just backlogs. Powered by AI coding agents.

---

## What is VelaraLoop?

VelaraLoop is a 6-phase framework for delivering digital products using AI coding agents as your execution team. It bridges the gap between traditional Product Ownership (planning only) and Agentic Engineering (coding only).

**For:** Product Owners, Product Managers, Founders, Solo builders, Consultants

**Core idea:** You define WHAT to build (specs, SoWs). AI agents build it. You ship, govern, and learn.

---

## The 6 Phases

```
DISCOVER → DEFINE → BUILD → SHIP → GOVERN → LEARN
    ↑                                           │
    └───────────────────────────────────────────┘
```

### 1. DISCOVER — Understand the problem

**Activities:**
- User research (interviews, surveys, data analysis)
- Market & competitor audit
- Problem statement definition
- Opportunity sizing & validation

**Frameworks used:** Double Diamond, JTBD, Lean Canvas, Opportunity Solution Tree

**Output:** Discovery Brief (1-pager)

**AI assists with:** Research summarization, competitor analysis, persona generation

---

### 2. DEFINE — Write the spec

**Activities:**
- Statement of Work (SoW) with phases & deliverables
- User stories (INVEST format)
- Architecture & technical decisions
- Prioritized backlog (RICE, MoSCoW, or Kano)
- Success metrics (OKRs, North Star Metric)

**Frameworks used:** Story Mapping, RICE, MoSCoW, OKRs, Kano Model, Spec-Driven Development

**Output:** SoW + Prioritized Backlog (ready for AI execution)

**AI assists with:** SoW drafting, user story generation, architecture diagrams

**This is where VelaraLoop diverges:**
> Traditional PO: writes stories → hands to dev team → waits weeks
> VelaraLoop PO: writes specs → gives to AI agent → ships same day

---

### 3. BUILD — Execute with AI agents

**Activities:**
- Open coding agent (Kiro, Claude Code, Cursor) with the spec
- AI generates code; you review and steer
- Test as you go (AI runs tests too)
- Commit with meaningful messages (reference SoW + phase)

**Frameworks used:** Sprint Planning, Definition of Done, Sprint Velocity

**Output:** Working code (committed, tested)

**Key skill:** Knowing WHAT to ask, not HOW to code.

**Delivery Modes:**
- **Solo Mode:** You + AI agent = you ship alone
- **Team Mode:** You + devs (who use AI) = you lead, they execute faster

---

### 4. SHIP — Deploy to production

**Activities:**
- Build & deploy (Vercel, Cloudflare, AWS, Docker)
- Verify: works on mobile? Performance? SSL?
- Domain + DNS setup
- Announce to stakeholders / users

**Frameworks used:** Release Checklist, Feature Flags, Canary/Blue-Green Deploys

**Output:** Live URL / Published product

**AI assists with:** Deploy commands, config troubleshooting, DNS setup

---

### 5. GOVERN — Track what was done

**Activities:**
- Write changelog (what was delivered, when, why)
- Update SoW status (phases → ✅)
- Version tracking (git commits with SoW references)
- Deploy history (dashboards, records)
- Kanban board updated

**Frameworks used:** Burndown/Burnup, Changelogs, Version Control, Kanban

**Output:** Changelog + SoW update + Deploy record

**Why this matters:**
> This is what NO bootcamp teaches. Real products need governance.
> Who did what, when, why. VelaraLoop makes it automatic, not bureaucratic.

---

### 6. LEARN — Measure and iterate

**Activities:**
- Check metrics (views, signups, revenue, engagement)
- Gather user feedback
- Run experiments (A/B tests, feature flags)
- Sprint retrospective
- Decide: iterate current? Or discover new?

**Frameworks used:** OKR Review, Pirate Metrics (AARRR), NPS, PMF Test, Retention Curves

**Output:** Learnings doc + Next sprint priorities

**The loop closes:** LEARN feeds back into DISCOVER (new problems found) or DEFINE (refined backlog).

## Delivery Modes

### Solo Mode
You are the Product Owner AND the builder. AI agents are your dev team.

Best for: Founders, freelancers, career switchers, consultants, personal projects.

### Team Mode
You are the Product Owner leading developers who use AI agents.

Best for: B2B consulting, startups, companies with AI-augmented engineering teams.

**In Team Mode, VelaraLoop PO:**
- Writes specs that devs+AI execute efficiently
- Reviews AI-generated PRs (knows what to look for)
- Adapts sprint velocity to AI-augmented pace (3-10x faster)
- Maintains governance when the team ships daily

## Getting Started

1. **Pick a product** (can be small: landing page, tool, API)
2. **DISCOVER:** Write a 1-page brief (problem, users, opportunity)
3. **DEFINE:** Write a SoW (use our template) with phases & deliverables
4. **BUILD:** Open your AI agent, paste the spec, start building
5. **SHIP:** Deploy it live (Vercel, CF Pages, etc.)
6. **GOVERN:** Write a changelog, update your SoW
7. **LEARN:** Check if it worked. What's next?

**Time to first loop:** 1 day (for a landing page) to 1 week (for an MVP)
