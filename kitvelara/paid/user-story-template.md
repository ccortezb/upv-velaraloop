## Story

**As a** [user type],
**I want to** [action/goal],
**So that** [benefit/value].

---

## INVEST Checklist

- [ ] **I**ndependent (can be built alone)
- [ ] **N**egotiable (details can flex)
- [ ] **V**aluable (delivers user value)
- [ ] **E**stimable (team can size it)
- [ ] **S**mall (fits in 1 sprint)
- [ ] **T**estable (clear pass/fail)

---

## Acceptance Criteria

**Given** [context/precondition],
**When** [action performed],
**Then** [expected result].

**Given** [context],
**When** [action],
**Then** [result].

## Definition of Done

- [ ] Code implements all acceptance criteria
- [ ] Tests pass
- [ ] Reviewed (by human or AI)
- [ ] Committed with message: `feat(sow-XXX): [story title]`
- [ ] Deployed or ready to deploy
