## What is VelaraLoop?

A 6-phase framework for delivering digital products using AI coding agents as your execution team. It bridges the gap between traditional Product Ownership (planning only) and Agentic Engineering (coding only).

```
DISCOVER → DEFINE → BUILD → SHIP → GOVERN → LEARN
    ↑                                           │
    └───────────────────────────────────────────┘
```

**For:** Product Owners, Product Managers, Founders, Solo builders, Consultants

---

## Two Modes

| Mode | You are... | AI is... | Best for |
|------|-----------|---------|----------|
| **Solo** | PO + builder | Your dev team | Founders, freelancers, consultants |
| **Team** | PO + leader | Your team's copilot | B2B, startups, companies |

## Quick Start

1. **Pick a product** (landing page, tool, API — start small)
2. **DISCOVER:** Write a [discovery brief](templates/discovery-brief.md)
3. **DEFINE:** Write a [SoW](templates/sow-template.md) with phases
4. **BUILD:** Open AI agent → paste spec → build
5. **SHIP:** Deploy live
6. **GOVERN:** Write a [changelog](templates/changelog-template.md)
7. **LEARN:** What worked? What's next?
